IP, 16진수 주소 사용 여부

일반적으로 웹사이트는 쉽게 기억할 수 있는 도메인 이름을 사용합니다. IP 주소는 숫자로 이루어져 있어 사용자들에게 익숙하지 않습니다.

비정상적으로 긴 URL

피싱 사이트는 긴 URL을 사용하여 사용자를 혼란시키고, URL에 악성 코드나 피싱 요소를 숨길 수 있습니다. 또한, 긴 URL을 통해 사용자가 URL 전체를 제대로 확인하지 못하게 합니다.

단축 URL의 실제 리다이렉션 주소

피싱 공격자는 단축 URL을 사용하여 사용자가 악성 사이트로 리다이렉션되는 것을 숨길 수 있습니다. 사용자는 단축 URL이 안전하다고 생각할 수 있지만, 실제로는 피싱 사이트로 연결될 수 있습니다.

https 프로토콜 사용 여부

피싱 사이트는 종종 HTTPS를 사용하지 않거나, 위조된 인증서를 사용하여 보안이 취약합니다. HTTPS가 없는 사이트는 사용자의 데이터를 안전하게 보호하지 못할 가능성이 큽니다.

도메인의 유효, 생성일자

피싱 사이트는 종종 새로운 도메인을 사용하여 빠르게 사이트를 생성하고 운영합니다. 도메인이 최근에 생성되었거나, 유효하지 않은 경우 피싱 사이트일 가능성이 높습니다.

비표준포트 사용 여부

피싱 사이트는 비표준 포트를 사용하여 탐지를 피하거나, 악성 트래픽을 전송할 수 있습니다. 비표준 포트를 사용하는 사이트는 의심을 받을 수 있습니다.

비슷한 문자에 대한 숫자의 대체

사용자가 정상적인 사이트로 착각하게 만들기 위해 문자 대신 숫자를 사용하여 도메인을 위조합니다. 이는 사용자가 URL의 미세한 차이를 쉽게 인식하지 못하게 하여 피싱 공격을 성공시키려는 의도입니다.
