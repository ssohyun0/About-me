Pay1oad 공식 웹사이트 제작

2024.03 ~ 06

📲 랜딩페이지

피그마를 이용하여 ui/ux를 디자인하였습니다.
깔끔하고 잘보이게 하기 위한 직관적인 ui 제작을 목표로 했습니다.
또한 랜딩페이지가 부드럽게 넘어갈 수 있도록 스크롤 페이지로 구현하였습니다.

🌐 JTW

개인키를 통해 통해 서명되기 때문에 안전하여 jwt 방식을 사용하였습니다.
accesstoken을 발급받는 형식으로 제작되었으나, 시간상의 이유로 refresh token은 아직 미구현 되었습니다.

작동 방식

1. 클라이언트에서 아이디와 비밀번호를 입력한다.
2. 입력한 아이디와 비밀번호를 서버로 전송한다.
3. 서버는 유효한 사용자인지 판단하고 올바르다면 accesstoken을 발급한다.
4. 토큰을 확인하고 저장한다.

😈 SQL Injection

주요 해킹 공격 기법중인 SQL Injection 공격을 방지하기 위한 부분을 추가하였습니다.
모든 입력창에서 sql 문법과 주요 특수문자들의 입력을 방지하였습니다.
sql 쿼리에서 사용되는 단어가 포함되거나, 특수문자의 경우 정규식을 이용하여 이를 감지하였습니다.
